| **캐릭터챗 유료화(슈퍼챗)** | 2024년 10월 도입 → **1개월 만에 월 매출 10억 돌파** |
| **크랙 분리 출시** | 캐릭터챗을 별도 앱 "크랙"으로 분리 → **유료화 1개월 만에 월 매출 20억 돌파** |

- 에이전트 확장 없이 캐릭터챗에 집중

### 2-3. 카카오톡 채널 API 5초 타임아웃 — 현실과 우회법

#### 문제의 본질
- 카카오톡 챗봇 스킬 API의 응답 제한 시간: **5초**
- LLM(GPT-4, Claude 등) 응답은 보통 5~30초 → **5초 내 불가능**
- 이 타임아웃은 **변경 불가** (카카오 공식 확인)

#### ✅ 공식 우회법: **콜백(Callback) API**

| 항목 | 내용 |
|------|------|
| **작동 방식** | 5초 안에 "처리 중" 응답 → 백그라운드에서 LLM 처리 → 콜백으로 결과 전송 |
| **승인 필요** | 챗봇 > 설정 > AI 챗봇 관리에서 신청. **영업일 1~2일** 소요 |
| **선행 조건** | "AI 챗봇"으로 전환 필요 (일반 챗봇에서 업그레이드) |
| **성공 사례** | AskUp(Upstage)이 이 방식으로 50만+ 유저 확보 |
| **UX** | 사용자에게 "답변 생성 중..." 같은 대기 메시지 → 수초 후 답변 도착 |

#### AskUp 성공 사례
- 2023년 3월 론칭 → **3일 만에 채널친구 3만명**
- 이후 **50만명 돌파** (OCR + ChatGPT 결합)
- 콜백 API로 5초 타임아웃 문제 해결 증명

**결론: 카톡 5초 타임아웃은 해결 가능한 문제. 콜백 API가 공식 솔루션이며, AskUp이 성공 사례.**

### 3-3. 소상공인 블로그 외주 시장 현실

#### 📊 시장 규모 & 가격대

| 항목 | 수치 |
|------|------|
| **한국 소상공인 수** | 약 680만 (2024 기준) |
| **블로그 관리 대행 가격** | 월 30~100만원 (포스팅 10~20건) |
| **건당 대필 비용** | 1건 약 3~5만원 (대행사), 1~2만원 (프리랜서) |
| **종합 마케팅 대행** | 월 70만~400만원+ |
| **한국 디지털 마케팅 시장** | 2035년 USD 31.14B 전망 (CAGR 13.6%) |

#### 실제 수요 증거
- **숨고** 플랫폼에 "블로그 마케팅" 카테고리 별도 존재 → 수요 입증
- **아이보스** 커뮤니티에서 대행사 홍보 활발 → B2B 마케팅 시장 활성
- 소상공인 월 마케팅 예산: **100만~150만원** 수준이 일반적
- **블로그 1건 = AI로 1,000~2,000원에 생산 가능** → 마진 10~50배
