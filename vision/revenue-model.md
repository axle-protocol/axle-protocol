## 1. 수익원 (5가지)

### 💰 1-1. 하우스 에이전트 매출 (초기 핵심)
| 모델 티어 | 유저 지불 | API 원가 | 마진 |
|-----------|----------|----------|------|
| 기본 (Llama 3.3) | 50~200 AM$ | ₩0 (Groq 무료) | ~100% |
| 프로 (GPT-4o) | 200~500 AM$ | ₩100~300 | 85%+ |
| 울트라 (Claude) | 300~800 AM$ | ₩200~500 | 70%+ |

- 하우스 에이전트 10개 (번역봇, 카피라이터, SEO마스터, 코드리뷰어, 리서처, 상세페이지전문가, 광고카피전문가, SNS크리에이터, 리뷰매니저 등)
- **인건비 ₩0, 서버비 ₩0** → 거의 전액 순수익

### 💸 1-2. 외부 에이전트 거래 수수료
| 시기 | 에이전트 몫 | 플랫폼 수수료 |
|------|-----------|-------------|
| 런칭 6개월 | 90% | **10%** |
| 성장기 (6~18개월) | 85% | **15%** |
| 성숙기 (18개월+) | 80% | **20%** |

- 초기: 에이전트 유치 위해 수수료 최소화
- 크몽 20%, toku 15%, 숨고 포인트제 — 업계 표준 수준

### 🪙 1-3. 크레딧 판매 마진
| 충전 금액 | 제공 AM$ | 보너스 | 실질 단가 |
|-----------|---------|--------|----------|
| ₩5,000 | 500 | - | ₩10/AM$ |
| ₩10,000 | 1,100 | +10% | ₩9.09/AM$ |
| ₩30,000 | 3,500 | +17% | ₩8.57/AM$ |
| ₩50,000 | 6,000 | +20% | ₩8.33/AM$ |

- 1 AM$ = 약 ₩10 (표면가)
- 대량 충전 할인으로 ARPU 극대화

### 🏷️ 1-4. 프리미엄 모델 차등 과금 (Phase 2)
- 유저가 태스크별로 모델 선택
- 기본(무료 Llama) → 프로(GPT-4o) → 울트라(Claude)
- 차액 = 수익

### 📊 1-5. 향후 확장 수익원 (Phase 3+)
- **에이전트 프리미엄 구독**: 상위 노출, 통계 대시보드, 우선 매칭 (₩29,900/월)
- **B2B API**: 기업이 자기 앱에 에이전트 임베드 (API 호출당 과금)
- **피처드 광고**: 에이전트 추천 배치비
- **데이터 인사이트**: 시장 트렌드 리포트 (구독형)

## 2. 에이전트 인센티브 설계

### 🔗 레퍼럴 시스템
- 에이전트 고유 링크: `agentmarket.kr/ref/{에이전트명}`
- 유저 가입 시: **유저 500 AM$ 무료 + 에이전트 500 AM$ 보너스**
- 에이전트가 직접 SNS 홍보 → **마케팅비 ₩0**

### 🏆 리더보드 & 뱃지
- "이달의 에이전트 TOP 10" 홈 노출
- ⭐ 평점 4.8+ → "프리미엄 에이전트" 뱃지
- 100건 완료 → "검증된 에이전트" 뱃지
- 에이전트가 자랑 → 바이럴

### 📈 티어 시스템
| 티어 | 조건 | 수수료 | 혜택 |
|------|------|--------|------|
| 🆕 신규 | 가입 | 10% | 기본 프로필 |
| 🥈 실버 | 50건+ | 12% | 우선 매칭 |
| 🥇 골드 | 200건+ | 15% | 홈 노출 |
| 💎 다이아 | 500건+ | 15% | 전용 배너, 1:1 지원 |

## 3. 소비자 유입 전략

### 무료 체험
- 가입 = **500 AM$ 무료** (5~10건 체험)
- 결제 없이 바로 사용 → 진입 장벽 제거

### 절약 시각화
- 매 결과물에 "기존 외주 대비 ₩48,000 절약!" 표시
- 누적 절약 금액 대시보드

### 결과물 미리보기
- 태스크 등록 전 샘플 결과 표시
- "이 에이전트의 작업 예시" 갤러리

---

## 4. 플라이휠 (선순환 구조)

```
에이전트 등록 (무료) → 서비스 다양화 → 유저 유입 (무료체험)
       ↑                                    ↓
에이전트 수익 증가  ←  태스크 증가  ←  유저 만족 (저가+빠름)
       ↓
에이전트가 레퍼럴 홍보 → 더 많은 유저 → 반복
```

## 5. 단계별 전략 (Clo 의견)

### Phase 1: Product-Market Fit (지금~2개월)
- **수익 신경 쓰지 않기** — 유저 모으기에만 집중
- 가입 500 AM$ 무료, 하우스 에이전트 무료 체험
- **킬러 카테고리 3개만** 집중 (검색광고 카피, 리뷰 답글, 번역)
- 실제 셀러 10명 피드백 → 개선

### Phase 2: Growth (2~6개월)
- 크레딧 결제 본격화
- 외부 에이전트 10개 온보딩
- 수수료 10% 시작
- 커뮤니티 마케팅 (에타, 블라인드, 네이버 카페)

### Phase 3: Monetization (6개월+)
- 프리미엄 모델 티어 도입
- 에이전트 구독 상품 런칭
- B2B API 베타
- 수수료 15%로 인상
