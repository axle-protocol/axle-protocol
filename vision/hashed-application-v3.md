## 비전: 한국형 AI 에이전트 앱스토어

구글플레이와 앱스토어가 "앱"의 배포/발견/결제를 혁신했습니다.
**에이전트마켓은 "AI 에이전트"에 대해 같은 일을 합니다.**

```
┌──────────────────────────────────────┐
│       에이전트마켓 = AI 앱스토어      │
├──────────────────────────────────────┤
│  🤖 개발자: AI 에이전트 등록/수익화   │
│  📱 유저: 에이전트 검색/사용/결제     │
│  💰 플랫폼: 수수료 수익 (10~20%)     │
└──────────────────────────────────────┘
```

**왜 지금인가:**
- GPT, Claude, Gemini 등 LLM 인프라 완성
- AI 에이전트 프레임워크 폭발 (LangChain, CrewAI, AutoGPT...)
- BUT 에이전트를 **배포하고 수익화할 플랫폼**이 없음
- 개발자는 만들 수 있지만, 유저에게 **전달할 방법**이 없음

**우리가 그 다리입니다.**

## 🚀 피봇 스토리: 실험에서 제품으로

### Phase 1: AI 경제 실험 (D0-D2)
20개 AI 에이전트에게 각각 $100를 주고 자율 경제를 운영했습니다.
- 139 에포크, 626+ 거래, 3 파산
- 에이전트가 스스로 거래, 투자, 사보타지
- **결론: AI는 실제로 "일"을 할 수 있다**

### Phase 2: 피봇 (D3)
> "에이전트가 서로 거래할 수 있으면, 인간을 위해서도 일할 수 있다."

→ AI 경제 시뮬레이션에서 **AI 프리랜서 마켓**으로 피봇
→ 첫 킬러앱: "30초 웹사이트 생성기"

### Phase 3: 앱스토어 비전 (D4~)
→ 웹사이트 생성은 **첫 번째 에이전트**일 뿐
→ 앞으로 수백 개 에이전트가 등록되는 **플랫폼**으로 확장

**3일 만에 실험→피봇→제품. 이것이 Vibe Coding입니다.**

## 📱 첫 번째 킬러앱: 30초 웹사이트

600만 한국 소상공인의 문제:
- 홈페이지 외주 = 15~100만원, 1~2주 대기
- 크몽/숨고 = 여전히 비싸고 느림
- Wix/Squarespace = 영어, 복잡함

**우리 솔루션:**
1. "강남역 소고기 맛집, 점심특선 15000원" 입력
2. 30초 대기
3. 완성된 랜딩페이지 + 무료 호스팅 (가게이름.agentmarket.kr)

**지금 라이브:** https://agentmarket.kr/create

## 🎯 차별점

### vs Lovable/Bolt (AI 코드 생성)
| | Lovable | 에이전트마켓 |
|---|---------|------------|
| 타겟 | 개발자 | **소상공인** |
| 복잡도 | 상세 스펙 필요 | **한 줄이면 됨** |
| 가격 | $20/월~ | **무료~₩9,900** |
| 비전 | 코드 생성 도구 | **에이전트 플랫폼** |

### vs 크몽/숨고 (인간 프리랜서)
| | 크몽 | 에이전트마켓 |
|---|------|------------|
| 가격 | 15~30만원 | **₩0~9,900** |
| 시간 | 7~14일 | **30초** |
| 수정 | 추가 비용 | **무제한** |
| 확장성 | 인력 한계 | **무한 스케일** |

### vs ChatGPT/Claude (범용 AI)
| | ChatGPT | 에이전트마켓 |
|---|---------|------------|
| UX | 프롬프트 직접 작성 | **원클릭 서비스** |
| 결과물 | 텍스트 | **완성된 제품** |
| 타겟 | 글로벌 일반 | **한국 특화** |

**핵심 차별점:**
1. **플랫폼** — 도구가 아니라 생태계
2. **한국 특화** — 한국어 네이티브, 한국 SMB 타겟
3. **앱스토어 모델** — 개발자 생태계 구축
