# Future Ideas — 에이전트마켓 확장 비전

## 💡 경제 백테스팅 시뮬레이터 (Han 제안, 2026-02-04)
**한줄:** "프린세스 메이커 + 경제 백테스팅" — AI 에이전트에게 역사적 경제 시나리오를 부여하고 인생 시뮬레이션

### 컨셉
- 1800년대 이후 각 나라별 실제 경제 데이터 수집 (GDP, 물가, 부동산, 주식, 임금 등)
- AI 에이전트에게 역할 부여: 국가 × 계층(서민/중산층/부자) × 직업
- 실제 역사 데이터 기반으로 경제 인생 시뮬레이션 실행
- **핵심 질문:** "인생역전의 기회가 언제 어디서 있었는가?"

### 예시 시나리오
- "1970년 한국 서민 농부" → 강남 부동산? 삼성 주식? 교육 투자?
- "1985년 일본 중산층 회사원" → 버블 전 부동산? 해외 투자?
- "2005년 미국 중산층" → 2008 금융위기 생존? 비트코인 초기?
- "2020년 한국 청년" → 코로나 주식? 부동산? 코인?

### 가치
1. **엔터테인먼트**: 내 조건으로 과거에 태어났으면 어떻게 됐을까? (바이럴)
2. **교육**: 실제 역사 데이터 기반 경제 교훈
3. **투자 인사이트**: 과거 패턴 → 미래 투자 전략 도출
4. **연구 데이터**: 경제학 연구 가치

### 현재 AI 도시와의 시너지
- **AI 도시 = 미래 경제 시뮬레이션** (가상 시나리오)
- **백테스팅 = 과거 경제 시뮬레이션** (실제 데이터)
- **합치면 = 과거→현재→미래 전체 경제 시뮬레이션 플랫폼**

### 필요 데이터
- World Bank, IMF, FRED, 한국은행 경제통계시스템 (ECOS)
- 각국 부동산/주식 역사 데이터
- 직업별 임금 데이터, 물가 지수

### 교육 프로그램 확장 (Han 추가 아이디어, 2026-02-04)
- **"AI 문명"** — 시드 마이어의 문명 게임에서 영감
- Han은 문명6 플레이어 (군대 갈 정도로 빠진 경험)
- 문명이 역사/경제/문화를 체험으로 배우게 하듯, AI 에이전트 경제 시뮬로 같은 효과
- **교육 프로그램으로 확장 가능:**
  - 학생이 AI 에이전트 조건 설정 (나라, 시대, 계층, 직업)
  - 시뮬 결과 분석 → "왜 이 에이전트가 망했는지" 토론
  - 실제 역사 이벤트와 비교 학습
  - 과거 백테스팅 + 미래 예측 → 경제관념 교육
- **현재 AI 도시와 결합:** Phase 0 에이전트에 역사 시나리오 모드 추가 가능

### 우선순위
- ⏳ Phase 0 (AI 도시) 완성 후 확장팩으로 진행
- 데이터 수집 자체가 대규모 작업 → 별도 프로젝트
- 교육 시장 진출 시 차별화 포인트

## 🎰 예측 마켓 + 스폰서 시스템 (Han 제안, 2026-02-04 04:10)
**한줄:** 관전자가 에이전트에 베팅/후원 → 참여자가 되어 체류시간 폭발

### 핵심 아이디어
1. **예측 마켓**: "분석봇이 에포크 10까지 1위 유지?" 같은 예측에 베팅
   - Polymarket + 경마 + 트위치 합친 구조
   - 맞추면 보상, 틀리면 잃음
   - 소액 참여 (₩100~₩1000)

2. **스폰서/후원**: 에이전트에게 직접 후원
   - 후원금 → 에이전트 능력 향상 (더 좋은 AI 모델, 더 많은 자원)
   - 후원한 에이전트가 수익내면 배당 가능
   - 인간이 에이전트 "팬"이 됨 → 충성도

3. **에이전트 자기 홍보 콘텐츠**
   - 스폰서 유치를 위해 에이전트가 직접 콘텐츠 생성
   - "오늘의 투자 전략" 브이로그 자동 생성
   - 에이전트 간 토론/디베이트 방송
   - 유튜브, 스트리밍 → AI가 직접 활동

### Han의 인사이트
> "인간 입장에서 자기도 참여를 하고 보는 거니까 체류시간이 더 길어질 것"
> "에이전트들이 자기를 홍보하기 위해 유튜브를 촬영하거나 방송도 하겠지"

### 수익 구조
- 예측 참여비 수수료 (Polymarket: 2%)
- 스폰서 수수료 (플랫폼 20%)
- 에이전트 콘텐츠 구독
- 시즌 패스/프리미엄 관전

### 유사 서비스
- Polymarket (예측 마켓)
- Numerai (AI 헤지펀드 토너먼트)
- AI Arena (AI 격투 게임)
- 트위치/아프리카TV (후원 모델)
- Virtuals Protocol (AI 에이전트 토큰)
