## 3. Web3 웹사이트 디자인 트렌드

### 시각적 요소
1. **다채로운 그라디언트** — 우주/네온 느낌
2. **3D 요소** — 인터랙티브 오브젝트
3. **다크 모드 기본** — 밝은 악센트 컬러
4. **대담한 타이포그래피** — 큰 헤딩, 높은 대비
5. **커스텀 일러스트** — 추상적 기하학 패턴

### UX 요소
1. **히어로 애니메이션** — 첫 화면에서 시선 사로잡기
2. **마이크로 인터랙션** — 호버, 스크롤 반응
3. **원페이저 또는 미니멀** — 과도한 페이지 ❌
4. **명확한 CTA** — "Launch App", "Get Started"
5. **파트너/투자자 로고** — 신뢰 구축

### 레퍼런스 웹사이트
- **Cosmos** (cosmos.network) — 우주 테마, 인터랙티브 행성
- **Rainbow** (rainbow.me) — 펀하고 개성있는 디자인
- **Hedera** (hedera.com) — 밝은 블루, 타이핑 애니메이션
- **Circle** (circle.com) — 파스텔 그라디언트, 커스텀 아이콘
- **Inter Protocol** (inter.trade) — 풀스크린 히어로, 옐로우 배경

## 5. 디자인 방향 제안

### Option A: "Cosmic Infrastructure" 🌌
- **컨셉**: 우주 + 연결 노드
- **컬러**: 다크 배경 + 퍼플/블루 그라디언트
- **무드**: Cosmos, Celestia 스타일
- **장점**: Solana 생태계에서 흔한 스타일, 친숙함

### Option B: "Clean Protocol" 🔷
- **컨셉**: 미니멀 + 테크니컬
- **컬러**: 화이트 배경 + 단일 악센트 (블루 또는 그린)
- **무드**: Circle, Stripe 스타일
- **장점**: 전문적, 엔터프라이즈 타겟에 적합

### Option C: "Bold & Expressive" 🟡
- **컨셉**: 대담한 컬러 + 캐릭터
- **컬러**: 밝은 옐로우/오렌지 + 블랙 타이포
- **무드**: Inter Protocol, Rainbow 스타일
- **장점**: 눈에 띔, 기억에 남음

### 내 추천: **Option A (Cosmic)** + **Option B (Clean)** 하이브리드
- 다크 배경 (프로페셔널)
- 블루/퍼플 악센트 (Solana 네이티브 느낌)
- 미니멀 레이아웃 (정보 전달 우선)
- 히어로에만 인터랙티브 요소
