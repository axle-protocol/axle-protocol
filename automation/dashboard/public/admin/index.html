Admin Setup

사장님 / 상품 / 매핑 / 샘플주문

Setup Instagram Queue Vendor
1) 사장님(vendor) 생성
데모용: username은 중복 불가. 비번은 변경/초대링크는 다음 스텝.
2) 상품 CSV 업로드

필수 컬럼: 상품번호(스마트스토어), 상품명 (나머지는 무시)

3) 매핑 (사장님 ↔ 상품번호)
검색 후 체크 → 저장
4) 주문 엑셀 업로드 (SmartStore 주문조회 xlsx)

암호화된 xlsx를 업로드하면 파싱해서 orders.json에 적재(상품번호 기반 vendor 자동배정)

주의: 이 파일은 주소/전화가 포함되지 않을 수 있음(현재 파서 기준).
5) 샘플 주문 생성(테스트)

선택한 vendor에게 임의 상품 주문 1건 생성

MVP: JSON 저장소 기반. 이후 SQLite + 초대링크/락아웃/감사로그 강화.
