Setup Instagram Queue Vendor
1) 사장님(배송 담당) 추가
사장님이 접속해서 자기 주문을 보고 송장을 입력합니다.
예: 아이디=vendor01 / 비번=임시비번(사장님에게 전달). 추후 초대링크로 개선 예정.
2) 상품목록 업로드(CSV)

필수 컬럼: 상품번호(스마트스토어), 상품명 (나머지는 무시)

3) 상품→사장님 배정(자동분류 기준)
주문 엑셀에 있는 ‘상품번호’로 자동 배정합니다. 검색 후 체크 → 저장
4) 주문 엑셀 업로드(SmartStore 주문조회)

암호화된 xlsx를 업로드하면 파싱해서 orders.json에 적재(상품번호 기반 vendor 자동배정)

주의: 이 파일은 주소/전화가 포함되지 않을 수 있음(현재 파서 기준).
6) 발송처리 엑셀 다운로드(스마트스토어 업로드)

송장 입력된 주문만 포함. 필요하면 분할 다운로드(chunk/size).

5) 미분류 주문 배정(사장님 미지정)

주문 업로드 후 vendorId가 비어있는 주문을 선택해서 vendor로 일괄 지정

7) 샘플 주문 생성(테스트)

선택한 vendor에게 임의 상품 주문 1건 생성

MVP: JSON 저장소 기반. 이후 SQLite + 초대링크/락아웃/감사로그 강화.
