# SOME — 한국 시장 리서치 (카톡/썸 답장 코치) v1

작성일: 2026-02-08

## TL;DR
- 한국에서 **“카톡 썸/연애 답장 코치”** 니즈는 분명히 존재하지만, 현재는 **제품**보다 **프롬프트/팁 콘텐츠**로 해결되는 경우가 많다.
- 이미 시장에는 **(1) 카카오톡 자체 AI 기능(실험실)**, **(2) OS/키보드 레벨 답장 추천**, **(3) 카톡 자동응답 봇(알림/Wear OS 우회)** 등이 존재한다.
- 따라서 SOME의 웨지는 **답장 생성 그 자체가 아니라**:
  - Training(승인 20회) → Autopilot Window(시간창)
  - 텔레그램 관제(로그/STOP/PAUSE)
  - 로컬 실행 + 서버 저장 0 기본
  - 한국형 말투/토글(ㅋㅋ/ㅎㅎ/이모지/줄바꿈)
  로 “신뢰 가능한 자율주행”을 파는 것.

## 2) 예상 시장 반응 (한국)

### 긍정 요인(수요)
- 답장/톡 센스는 명확한 페인포인트.
- 프롬프트 공유가 돌아다닌다는 건 “제품이 없어서 프롬프트로 때우는” 상황일 수 있음.
- OS/카톡 자체 AI 기능으로 ‘추천 답장’ UX가 이미 침투 중.

### 부정 요인(거부감)
- “대신 보내기(자율주행)”는 사칭/사고/오발송 불안이 큼.
- 그래서 **신뢰 UX**가 없으면 반감이 커질 수 있음.

SOME의 방어 장치(필수):
- Training(승인 20회)로 신뢰 형성
- Autopilot Window(시간창)로 범위 제한
- 텔레그램 관제(로그/STOP/PAUSE)
- 로컬 실행 + 서버 저장 0 기본

## 4) Opus 참고용 코멘트
- 한국 시장은 ‘추천 답장’ 자체는 익숙해질 수 있으나, 자동 전송은 거부감이 큼.
- 따라서 문장 생성 품질만큼 **안전/완곡 표현/금칙 처리**가 중요.
- 토글(ㅋㅋ/ㅎㅎ, 이모지, 줄바꿈, 오타 유지)은 “한국 감성”에서 체감이 큼.
