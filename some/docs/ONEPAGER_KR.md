# SOME — 1페이지 (한국형 Second-Me / 카톡 썸 대화 자율주행)

작성일: 2026-02-08

## 한 줄
**SOME = 카톡/DM 대화를 “내 말투로” 끊기지 않게 이어주는, 관제 가능한 자율주행(시간창) 비서**

---

## 문제(페인)
- 답장 센스/타이밍 때문에 썸이 깨진다.
- 운동/운전/회의 중에 대화가 끊기면 흐름이 끝난다.
- ChatGPT 프롬프트/키보드 추천은 “복붙” 수준이라 번거롭고, 맥락이 계속 이어지지 않는다.
- ‘대신 보내기’는 무섭다(사칭/오발송/선 넘기/금전·개인정보 리스크).

---

## 해결(제품)
### 1) Training Mode → 신뢰 확보
- 처음엔 **승인 20회**까지는 메시지마다 확인 후 전송
- 유저가 “이 톤/이 기준이면 괜찮다”를 직접 검증

### 2) Autopilot Window → 끊김 제거
- **30~60분 같은 시간창** 동안만 자율주행 ON
- 상대/채팅방 **화이트리스트**로 범위 제한
- 폰(텔레그램)에서 실시간 관제 + STOP

### 3) 한국형 말투/컨트롤
- 프리셋: **내말투 / 다정 / 쿨 / 공감**
- 토글: ㅋㅋ/ㅎㅎ, 이모지, 줄바꿈, 길이, 오타 유지, 플러팅 강도(0~3)

---

## 핵심 차별점(우리가 이기는 이유)
- ❌ 답장 생성기(복붙) vs ✅ **Second-Me Conversation Runtime(맥락 누적)**
- ✅ **관제 가능한 자율주행(시간창)** + STOP/PAUSE/로그
- ✅ **로컬 실행 + 서버 저장 0 기본**(한국 신뢰 포인트)
- ✅ 선 넘는 표현 차단(boundaryFilter) + 위험 시 자동 중지

---

## 안전/프라이버시(명시)
- 기본: 대화 원문은 **유저 기기 로컬에만** 저장(서버 저장 0)
- 텔레그램 관제: 기본은 **요약/마스킹**(원문 전체는 옵션)
- 금칙(예시): 돈/주소/계좌/링크/성적동의/욕설/협박 → 감지 시 **즉시 중지 + 알림**

---

## 과금(적자 방지 구조)
### 선택형
1) **BYOK(추천/기본값)**: 사용자가 자기 API Key로 비용 부담
2) **Hosted(편의성)**: 우리 키 제공 + **월 크레딧 상한**(초과 시 일시정지/추가결제)

### Hosted 모델 라우팅(비용/속도)
- cheap(필터/요약/라우팅): `gpt-4o-mini`
- 답장 생성(한국어/뉘앙스): Claude
- 병렬 실행 + 위험 시 전송 차단

---

## 가격 초안(테스트용)
- BYOK Basic: ₩9,900/월 (추천+관제)
- BYOK Pro: ₩29,900/월 (Autopilot Window)
- Hosted Pro: ₩69,900/월 (월 크레딧 포함, 상한)
- 셋업비: ₩49,000~₩99,000 (원클릭 설치 + “안 꺼지게 유지” 세팅)

---

## GTM(초기)
- 1차 타겟: 썸/연인 관계에서 답장 스트레스 큰 20~30대
- 채널: 지인/소개, 인스타/링크드인 DM(웹 자동화 데모), 커뮤니티
- 메시지: “답장 추천”이 아니라 **‘끊김 없는 대화 유지’ + ‘사고 방지’**

---

## Opus(Claude) 작업 포인트
- 한국어 뉘앙스/완곡 표현
- boundaryFilter(선 넘기 방지) 정책 + 대체 문장 패턴
- 프리셋/토글이 문장에 반영되는 룰표
- 고정 JSON 출력 스키마 + risk taxonomy
