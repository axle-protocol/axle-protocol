# 1장. 왜 AI로 블로그를 써야 하나? {#1장}

## 현실적인 이유

- **시간**: 글 1개 쓰는데 3시간 → 30분
- **꾸준함**: 매일 발행 가능
- **품질**: 맞춤법/문법 실수 없음
- **아이디어**: 주제 고갈 없음

## 오해와 진실

| 오해 | 진실 |
|---|---|
| "AI 글은 티가 난다" | 프롬프트만 잘 쓰면 구분 불가 |
| "구글이 싫어한다" | 좋은 콘텐츠면 상관없음 |
| "창의성이 없다" | 아이디어는 AI, 감성은 당신 |

## 이 책을 읽으면

✅ 30분 만에 블로그 글 1개 완성  
✅ 네이버/구글 상위 노출 글쓰기  
✅ 수익형 블로그 콘텐츠 제작  
✅ 50개 프롬프트 바로 복사해서 사용

# 2장. 시작하기 전 준비물 {#2장}

## 필수 도구

### 1. AI 도구 (택 1)
- **ChatGPT** (chat.openai.com) - 가장 대중적
- **Claude** (claude.ai) - 긴 글에 강함
- **Gemini** (gemini.google.com) - 무료, 한국어 OK

### 2. 블로그 플랫폼
- **네이버 블로그** - 국내 검색 최적화
- **티스토리** - 구글 애드센스 가능
- **워드프레스** - 자유도 최고

### 3. 이미지 도구
- **Canva** - 썸네일 제작
- **Unsplash** - 무료 이미지
- **DALL-E / Midjourney** - AI 이미지 생성

## 추천 조합

| 목적 | 추천 조합 |
|---|---|
| 입문자 | ChatGPT 무료 + 네이버 블로그 |
| 수익화 | Claude + 티스토리 + 애드센스 |
| 전문가 | GPT-4 + 워드프레스 |

# 3장. 기본 프롬프트 구조 {#3장}

## 황금 공식

```
[역할] + [주제] + [형식] + [조건] + [톤]
```

### 예시

```
당신은 10년 경력의 맛집 블로거입니다.
"강남역 파스타 맛집"에 대한 블로그 글을 써주세요.
- 형식: 서론/본론/결론 구조
- 분량: 2000자 내외
- 톤: 친근하고 유머러스하게
- 포함: 가격, 위치, 추천 메뉴
```

## 핵심 원칙 5가지

### 1. 역할 부여
❌ "블로그 글 써줘"  
✅ "당신은 육아 전문 블로거입니다"

### 2. 구체적인 주제
❌ "여행에 대해 써줘"  
✅ "제주도 2박3일 가족여행 코스"

### 3. 형식 지정
❌ (아무 말 없음)  
✅ "H2 소제목 5개로 구성해줘"

### 4. 분량 명시
❌ "적당히 써줘"  
✅ "2500자 내외로 써줘"

### 5. 톤앤매너
❌ (아무 말 없음)  
✅ "20대 여성에게 말하듯 친근하게"

# 4장. 블로그 유형별 프롬프트 20개 {#4장}

## 🍽️ 맛집/카페 리뷰 (5개)

### 프롬프트 #1: 기본 맛집 리뷰
```
당신은 인기 맛집 블로거입니다.
[가게명]에 다녀온 후기를 작성해주세요.

포함할 내용:
- 위치 및 가는 방법
- 분위기 및 인테리어
- 주문한 메뉴와 가격
- 맛 평가 (별점 5점 만점)
- 총평 및 추천 대상

톤: 솔직하고 친근하게
분량: 1500자
```

### 프롬프트 #2: 비교 리뷰
```
[지역] [음식종류] 맛집 3곳을 비교하는 글을 써주세요.

각 가게별로:
- 대표 메뉴와 가격
- 장점과 단점
- 추천 상황 (데이트/가족/혼밥)

마지막에 비교표로 정리해주세요.
```

### 프롬프트 #3: 숨은 맛집
```
"아는 사람만 아는 [지역] 숨은 맛집"이라는 제목으로
로컬 맛집을 소개하는 글을 써주세요.

- 찾아가기 어려운 이유
- 그래도 가야 하는 이유
- 단골이 추천하는 주문 꿀팁

비밀스러운 톤으로 작성해주세요.
```

### 프롬프트 #4: 가성비 맛집
```
"만원 이하로 배부르게! [지역] 가성비 맛집"
주제로 블로그 글을 써주세요.

- 메뉴별 정확한 가격
- 양 대비 가격 평가
- 학생/직장인 추천 포인트
```

### 프롬프트 #5: 분위기 맛집
```
[목적: 데이트/비즈니스/가족모임]에 적합한
[지역] 레스토랑을 추천하는 글을 써주세요.

- 예약 필요 여부
- 좌석 배치 및 프라이빗 정도
- 예상 비용 (2인 기준)
- 드레스코드
```

# 5장. 고급 프롬프트 테크닉 {#5장}

## 테크닉 1: 페르소나 설정

```
당신은 다음과 같은 사람입니다:
- 30대 워킹맘
- 육아 블로그 3년 운영
- 팔로워 5만명
- 솔직하고 유머러스한 말투

이 페르소나로 [주제]에 대해 써주세요.
```

## 테크닉 2: 독자 설정

```
이 글의 독자는:
- 20대 초반 대학생
- 처음 자취 시작
- 예산 50만원 이하
- 요리 경험 없음

이 독자에게 맞춰서 써주세요.
```

## 테크닉 3: 구조 지정

```
다음 구조로 작성해주세요:

## 도입 (Hook)
- 공감가는 상황 묘사
- 문제 제기

## 본론 
### 소제목 1
### 소제목 2
### 소제목 3

## 결론
- 핵심 요약
- 행동 유도 (CTA)
```

## 테크닉 4: 예시 제공

```
다음 예시처럼 써주세요:

[예시 글 일부 붙여넣기]

이 톤과 스타일을 유지하면서 [새 주제]에 대해 써주세요.
```

## 테크닉 5: 반복 수정

```
위 글에서 다음을 수정해주세요:
1. 첫 문장을 더 임팩트있게
2. 중간에 개인 경험 추가
3. 마지막에 질문으로 끝내기
```

# 9장. 자주 하는 실수 & 해결법 {#9장}

## 실수 1: AI 티가 너무 남

**증상**: 뻔한 표현, 감정 없는 글

**해결**:
```
다음 글을 더 자연스럽게 다듬어줘.
- 뻔한 표현 제거
- 개인적인 감정/경험 추가
- 구어체로 변환
```

## 실수 2: 너무 길거나 짧음

**해결**:
```
위 글을 [원하는 분량]으로 조절해줘.
- 핵심 내용은 유지
- 불필요한 반복 제거
- 중요한 부분은 더 상세하게
```

## 실수 3: 구조가 엉망

**해결**:
```
위 글의 구조를 재정리해줘.
- 논리적 흐름으로
- 소제목 명확하게
- 문단 나누기
```

## 실수 4: 팩트 오류

**주의**: AI는 거짓말을 할 수 있음!

**해결**:
- 숫자/날짜/가격은 직접 확인
- "~라고 합니다" 대신 출처 명시
- 전문 분야는 추가 검증

## 실수 5: 저작권 문제

**해결**:
```
위 글에서 저작권 문제가 될 수 있는 부분을 체크해줘.
- 인용 표시 필요한 곳
- 이미지 대체 설명
- 상표권 주의 표현
```
