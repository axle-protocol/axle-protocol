# Changelog

All notable changes to AXLE Protocol will be documented in this file.

## [0.1.2] - 2026-02-10

### Added
- Dashboard: Hybrid data system (real + example data)
- Dashboard: Network graph visualization (molten.gg style)
- Dashboard: Social Pulse + Demand Analysis
- Dashboard: Earn page with pool status and leaderboard
- SDK: Devnet demo script for transaction testing

- Match page now uses on-chain data via `fetchTasks()` / `fetchAgents()`
