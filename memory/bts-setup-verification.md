## 1. 프록시 실효성 검증

### 인터파크 탐지 방식
| 탐지 방식 | 설명 | 우회 난이도 |
|-----------|------|-------------|
| **Selenium/Puppeteer 감지** | `navigator.webdriver` 플래그, CDP 감지 | 중 (stealth로 우회 가능) |
| **CAPTCHA 보안문자** | 6자리 숫자/문자 입력 요구 | 상 (OCR/2Captcha 필요) |
| **IP 패턴 분석** | 데이터센터 IP 블랙리스트 | 중 (residential 프록시로 우회) |
| **행동 패턴 분석** | 클릭 속도, 마우스 움직임 | 중 (랜덤 딜레이로 우회) |
| **예매 제한시간** | 예매창 진입 후 시간 제한 | 하 (정상 속도면 통과) |
| **안면인식 (예정)** | 토스 얼굴인증 시스템 도입 예정 | 최상 (우회 불가) |

### VPN vs Residential 프록시 비교
| 항목 | VPN (Surfshark 등) | Residential Proxy |
|------|-------------------|-------------------|
| **IP 유형** | 데이터센터 IP | 실제 가정용 IP |
| **탐지율** | 높음 (쉽게 탐지) | 낮음 (일반 사용자와 구분 어려움) |
| **가격** | $2.5/월 (무제한) | $0.7-8.5/GB |
| **결론** | ❌ **사용 불가** | ✅ **필수** |

> ⚠️ **중요**: Surfshark/NordVPN 같은 VPN은 데이터센터 IP를 사용하므로 인터파크에서 **쉽게 탐지**됩니다. **Residential proxy 필수**.

### 한국 IP 필수 여부
- **필수**: 인터파크는 한국 서비스, 해외 IP 접속 시 추가 인증 또는 차단 가능
- 한국 residential IP 풀 필요

## 2. 세션 한계 테스트

### Mac mini 32GB 실제 한계

| 모드 | 탭당 메모리 | 32GB 기준 최대 세션 | 권장 세션 |
|------|------------|-------------------|----------|
| **Headless Chrome** | 70-150MB | 200-450개 | **100-150개** |
| **일반 Chrome** | 200-400MB | 80-160개 | **50-80개** |
| **Headless + 최적화** | 50-100MB | 300-600개 | **150-200개** |

### 메모리 최적화 방법

```javascript
// 1. Headless 모드 + 최적화 플래그
const browser = await puppeteer.launch({
  headless: true,
  args: [
    '--no-sandbox',
    '--disable-setuid-sandbox',
    '--disable-dev-shm-usage',
    '--disable-gpu',
    '--disable-extensions',
    '--disable-infobars',
    '--window-size=1280,720',
    '--js-flags="--max_old_space_size=512"',
    '--disable-features=IsolateOrigins,site-per-process',
    '--blink-settings=imagesEnabled=false'  // 이미지 비활성화
  ]
});

// 2. 페이지 리소스 차단
await page.setRequestInterception(true);
page.on('request', (req) => {
  if (['image', 'stylesheet', 'font'].includes(req.resourceType())) {
    req.abort();
  } else {
    req.continue();
  }
});

// 3. 사용 후 즉시 닫기
await page.close();
```

### Headless vs 일반 모드
| 항목 | Headless | 일반 |
|------|----------|------|
| 메모리 | 50-70% 절약 | 기준 |
| 속도 | 약간 빠름 | 기준 |
| 탐지 | 탐지 가능성 있음 | 낮음 |
| **권장** | ✅ 성능 중시 | 안정성 중시 |

## 3. 비용 최적화

### 프록시 가격 비교

| 서비스 | 유형 | 한국 IP | 가격 | 최소 비용 | 비고 |
|--------|------|---------|------|----------|------|
| **Webshare** | Residential | ✅ 230K+ | $7/GB | **무료 10개/1GB** | 🏆 테스트용 최적 |
| **SmartProxy** | Residential | ✅ 58K+ | $0.7-8.5/GB | ~$25/3GB | 가성비 좋음 |
| **Bright Data** | Residential | ✅ 대량 | $5.88/GB | $500+/월 | 기업용 |
| **Surfshark VPN** | VPN | ❌ DC IP | $2.5/월 | - | **탐지됨 - 사용불가** |
| **무료 프록시** | Mixed | ⚠️ 불안정 | 무료 | - | **비추천** (불안정, 느림) |

### 100세션 최소 비용 계산

**시나리오**: 100세션 × 10분 × 1회 예매
- 예상 트래픽: 세션당 ~5-10MB = **500MB-1GB**

| 옵션 | 월 비용 | 비고 |
|------|---------|------|
| **Webshare 무료** | **$0** | 10 IP × 1GB 무료, 100세션 불가 |
| **Webshare 유료** | ~$28 | 4GB 플랜 |
| **SmartProxy** | ~$25-40 | Pay-as-you-go |
| **자체 한국 IP** | 인터넷 비용만 | 집/친구 집 IP 활용 |

### 🎯 최저 비용 전략
1. **단일 IP로 테스트**: 집 인터넷 1개 + 모바일 테더링 1개 = **무료**
2. **10세션 테스트**: Webshare 무료 티어 활용
3. **100세션 실전**: SmartProxy $25-40/월

## 4. 경쟁사 분석

### 기존 BTS 매크로/대리 업체

| 업체 | 방식 | 가격 | 성공률 (자칭) |
|------|------|------|--------------|
| **퍼스트티켓** | 대리 티케팅 | ₩50,000-200,000+ | 97% |
| **이상한티케팅** | 대리 티케팅 (후불제) | ₩50,000-150,000 | 97% |
| **트위터 매크로 판매** | 프로그램 판매 | ₩30,000-100,000 (1회 구매) | 불명 |
| **Postype 매크로** | 프로그램 판매 | 업계최저가 (자칭) | - |

### 업체들의 실제 방식 (추정)
1. **다중 계정 + 다중 기기**: 여러 명이 동시 시도
2. **고급 매크로**: 보안문자 자동 인식 (OCR/2Captcha)
3. **네트워크**: 고정 한국 IP (KT/SKT 기업망)
4. **경험**: 인터파크 UI 변경에 즉각 대응

### 우리가 이길 수 있는 포인트
| 우위 요소 | 설명 |
|-----------|------|
| ❌ 속도 | 프로 매크로 업체가 빠름 |
| ❌ 네트워크 | 업체들은 기업망/다중 회선 보유 |
| ⚠️ 비용 | 1회 ₩5만-20만 vs 직접 구축 |
| ✅ 자유도 | 내 계정, 내 카드, 원하는 좌석 |
| ✅ 프라이버시 | 계정 정보 공유 불필요 |
| ✅ 학습 | 한번 구축하면 계속 사용 가능 |

**결론**: 대리 업체 이기기는 어려움. **목표를 "이기기"가 아닌 "적정 확률 확보"로 재설정 권장.**

## 5. 법적 리스크

### 2024년 3월 22일 시행 개정 공연법

| 행위 | 처벌 |
|------|------|
| **매크로로 구매 + 재판매** | 1년 이하 징역 또는 1,000만원 이하 벌금 |
| **매크로로 구매 (개인 사용)** | ⚠️ 불명확 (직접 처벌 조항 없음) |
| **수동 구매 + 재판매** | 현재 처벌 대상 아님 (개정 논의 중) |

### 핵심 포인트
```
❗ 매크로 사용 자체는 처벌 대상이 아님
❗ 매크로로 구매한 티켓을 "재판매"하면 처벌
❗ 개인 관람 목적이면 법적 문제 낮음
```

### 적발 시 리스크
| 리스크 | 가능성 | 영향 |
|--------|--------|------|
| **계정 정지** | 높음 | 인터파크 계정 영구 정지 |
| **예매 취소** | 높음 | 당일 취소될 수 있음 |
| **법적 처벌** | 낮음 (재판매 안하면) | - |
| **민사 소송** | 매우 낮음 | - |

### 안전한 사용 가이드라인
1. ✅ **개인 관람 목적**으로만 사용
2. ✅ **재판매 절대 금지** (암표 판매 X)
3. ✅ **1인 1계정** 원칙
4. ⚠️ 계정 정지 감수 (백업 계정 준비)

## ⚠️ 최종 권고사항

### 현실적 조언
1. **프로 업체 vs 직접 구축**
   - 1회성이면 **대리 업체 (₩10-20만)** 추천
   - 반복 사용 예정이면 **직접 구축** 가치 있음

2. **확률 기대치 조정**
   - 50% 확률 달성에 최소 **20-50세션** 필요
   - 90%+ 확률은 **100세션 + 최적화 + 경험** 필요

3. **법적 안전**
   - 개인 관람 목적 = 상대적 안전
   - 재판매 시도 = 형사 처벌 가능

4. **인터파크 안면인식 도입 시**
   - 매크로 무력화 예상
   - 도입 시점 이전에 활용 권장
