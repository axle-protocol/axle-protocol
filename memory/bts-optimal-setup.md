## 2. VPN vs 프록시 비교

### 2.1 상세 비교표

| 항목 | VPN | Datacenter 프록시 | Residential 프록시 |
|------|-----|-------------------|---------------------|
| **탐지율** | 60-80% | 70-90% | **5-15%** |
| **속도** | 빠름 | 매우 빠름 | 중간 |
| **비용** | $2-13/월 | $0.05-0.5/IP | $1.4-10/GB |
| **IP 수** | 수천 | 수만 | 수백만 |
| **추천도** | ⭐⭐ | ⭐ | ⭐⭐⭐⭐⭐ |

### 2.2 VPN 옵션 (권장하지 않음)

| 서비스 | 가격 | 한국 서버 | 탐지 우회 | 평가 |
|--------|------|-----------|-----------|------|
| Surfshark | $1.99/월 (2년) | ✅ | ❌ 낮음 | ⚠️ |
| NordVPN | $2.99/월 (2년) | ✅ | ❌ 낮음 | ⚠️ |
| ExpressVPN | $6.67/월 (1년) | ✅ | ❌ 낮음 | ⚠️ |

**VPN 비추천 이유:**
- 티켓팅 사이트가 VPN IP 대역 블랙리스트 보유
- 같은 IP를 수천 명이 공유 → 의심 패턴
- Dedicated IP 옵션도 datacenter IP라 탐지됨

### 2.3 Residential 프록시 (강력 추천)

| 서비스 | 가격/GB | 한국 IP | Pay-as-you-go | 추천도 |
|--------|---------|---------|---------------|--------|
| **Smartproxy** | $0.7/GB | ✅ | ✅ | ⭐⭐⭐⭐⭐ |
| **Webshare** | $1.4/GB | ✅ | ✅ | ⭐⭐⭐⭐ |
| IPRoyal | $4.5/GB | ✅ | ✅ | ⭐⭐⭐ |
| Oxylabs | $4-7.5/GB | ✅ | ✅ | ⭐⭐⭐ |

### 2.4 티켓팅 데이터 사용량 추정

```
세션당 예상 데이터:
- 페이지 로드: 2-5MB
- 대기열 대기 (30분): 0.5-1MB
- 좌석 선택 + 결제: 1-2MB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
세션당 총: 약 5-8MB

10세션 x 8MB = 80MB ≈ 0.1GB
20세션 x 8MB = 160MB ≈ 0.2GB
```

**결론: 티켓팅 1회당 0.1-0.3GB 사용 → $1-3 비용**

## 3. 계정 전략

### 3.1 5계정 최적 분배

```
계정 배치 전략:

계정1 (메인)    → 3-4 세션  → VIP만 타겟
계정2 (서브1)   → 2-3 세션  → VIP + A석
계정3 (서브2)   → 2-3 세션  → VIP + A석
계정4 (백업1)   → 1-2 세션  → 모든 좌석

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
총: 10-14 세션
```

### 3.2 계정당 세션 한계

| 플랫폼 | 권장 세션/계정 | 최대 | 주의사항 |
|--------|----------------|------|----------|
| 인터파크 | 2-3 | 5 | 동일 IP 금지 |
| Ticketmaster | 1-2 | 3 | 매우 엄격 |
| Weverse Shop | 2-3 | 4 | 상대적 관대 |

### 3.3 차단 위험 관리

**필수 조치:**
1. ✅ 각 계정별 다른 프록시 IP
2. ✅ 각 계정별 다른 브라우저 fingerprint
3. ✅ 계정별 다른 결제수단 (카드)
4. ✅ 계정별 다른 이름/주소 (가족/친구)
5. ✅ 자연스러운 클릭 패턴 (딜레이 추가)

**위험 행동:**
- ❌ 동일 IP에서 다중 계정
- ❌ 너무 빠른 클릭 (100ms 이하)
- ❌ 동일 fingerprint로 다중 세션
- ❌ 페이지 과다 새로고침

```
1. smartproxy.com 가입
2. Residential Proxies → Pay As You Go → $7/GB 선택
3. 3GB 구매 ($21)
4. Dashboard → Residential → Endpoint Generator
5. 설정:
   - Location: South Korea
   - Session type: Rotating (또는 Sticky 30분)
   - Protocol: HTTP/HTTPS
6. 생성된 프록시 정보 저장:
   - Host: kr.smartproxy.com
   - Port: 10000
   - Username: user-xxx
   - Password: yyy
```

#### Step 2: 안티디텍트 브라우저 (GoLogin)

```
1. gologin.com 가입 (Google/이메일)
2. 앱 다운로드 (macOS)
3. 새 프로필 생성:
   - 이름: BTS_1, BTS_2, BTS_3
   - OS: macOS
   - Browser: Chrome 최신
   - Proxy: 각각 다른 Smartproxy 세션

4. 프로필당 인터파크 로그인 + 쿠키 저장
```
