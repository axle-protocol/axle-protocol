# BTS v5.5.0 - Cycle 3 Review

## 버전: 5.4.0 → 5.5.0

## 수정된 이슈

### 🔴 좌석 선택 개선

1. **픽셀 분석 색상 범위 확장**
   - 문제: 녹색 감지가 너무 제한적 (g > 150 조건)
   - 해결: 다양한 녹색 톤 감지 (진한 녹색 포함)
   - 추가: 노란색/금색 특별석 감지

2. **좌석 우선순위 개선**
   - 문제: type만으로 정렬, 동일 type 내 순서 무작위
   - 해결: score 기반 정렬 추가 (색상 강도 높은 좌석 우선)

3. **_complete_selection 에러 복구**
   - 문제: 한 번 실패 시 바로 종료
   - 해결: max_retries=3으로 재시도, 에러 메시지 감지

### 🟡 안정성 개선

4. **_check_selection_error 함수 추가**
   - 새 기능: 좌석 선택 에러 메시지 감지
   - 감지 항목: 매진, 시간 초과, 이미 선택된 좌석 등

5. **step_select_seat 로깅 개선**
   - 문제: 남은 시도 횟수 표시 없음
   - 해결: remaining 변수로 명확히 표시
