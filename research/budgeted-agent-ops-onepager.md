# Budgeted Agent Ops (AXLE × OpenClaw) — 1페이지 제안서

## 한 줄
**에이전트가 ‘일’을 할수록 생기는 2가지 공포(비용 폭주, 사고/책임)**를 **예산 가드레일 + 영수증(증거) + 에스크로 정산**으로 묶어서 해결하는 운영형 제품.

## 누구에게 파나 (ICP)
1) **개발팀/스타트업(한국+해외)**: PR 리뷰, 릴리즈, 보안 스캐닝이 병목인데 운영할 사람이 없음
2) **SMB 오퍼레이터(채널 기반 운영)**: 텔레그램/슬랙으로 업무를 받고 처리 내역을 남겨야 함
3) **에이전트 파워유저(OpenClaw 사용자)**: 크론/하트비트 돌리다 비용/보안 리스크로 멈춤

## 해결하는 Pain (근거 기반)
- 비용 폭주: 크론/하트비트/컨텍스트로 토큰이 예측 불가하게 샘
- 신뢰/보안: 브라우저/메일/결제 같은 “실행”은 사고나면 끝
- 운영 부재: PR봇은 댓글만 달고, SLA/설정 유지/야간 대응은 안 해줌

## 제품 구성 (3 primitives)
1) **Budget (예산 가드레일)**
- scope별 한도: `cron/job/channel/tool/repo` 단위
- 루프 감지 → 자동 pause + 승인 요청
- 월 예상비용 초과 예측 시 사전 차단

2) **Receipts (영수증/증거)**
- 작업마다: 입력/출력 요약 + 아티팩트 해시(스크린샷/URL/patch/diff)
- “무엇을 했는지” 재현 가능한 로그

3) **Settlement (정산)**
- 에스크로(착수/중도/완료) + 승인 기반 release
- (옵션) 성과기반: 머지/버그픽스/리포트 승인 등 조건

## 첫 번째 웨지(팔리는 유스케이스 1개)
**Managed 24/7 Review Ops**
- PR 뜨면 1차 리뷰/리스크 체크/테스트 가이드 + 릴리즈 노트 자동
- SLA: 예) 15분 내 첫 응답, 6시간 내 전체 리뷰, 고위험은 escalations
- “댓글봇”이 아니라 “리뷰 운영”을 팔기

## 가격(초안) — 한국식/글로벌식 둘 다 가능
### 한국식 (빠른 매출)
- 세팅비: ₩990,000~₩2,900,000
- 월: ₩290,000~₩990,000
- 추가 과금(선택): 활동/작업당(채널톡처럼)

### 글로벌식 (확장)
- $199~$499 / month + usage (runs/activities/agent-hours)

## 경쟁 대비 차별점
- CodeRabbit/Qodo/Bito 등: 소프트웨어(댓글) 제공 → 운영/보장 없음
- 우리는: **예산 통제 + 증거 + SLA 운영**으로 “결과”를 보장
- OpenClaw: 실브라우저/채널/스케줄 러ntime
- AXLE: 정산/평판/증거 레일

## 7일 MVP 목표
- 데모 2개:
  1) 예산 캡 걸린 daily ops 에이전트(자동 pause/승인)
  2) 에스크로 위임→납품(증거)→정산
- 설치/온보딩 5분: 템플릿 + 문서
