## 🛠️ 기술 스택 (확정)

### 1. Camoufox (메인 브라우저)
```bash
pip install camoufox[geoip] playwright
playwright install firefox
```

**장점:**
- C++ 레벨 핑거프린트 변조 (JS 감지 불가)
- Firefox 기반 (Chrome보다 덜 감시됨)
- 매 실행마다 자동 랜덤 핑거프린트
- Cloudflare Turnstile 우회 ✅

**검증:** GitHub ⭐ 5.2k, 활발한 업데이트

### 2. 2Captcha API
```bash
pip install 2captcha-python
```

**가격표:**
| CAPTCHA 종류 | 가격 (1000개당) |
|-------------|----------------|
| 일반 이미지 | $0.50~$1.00 |
| reCAPTCHA v2 | $2.99 |
| Turnstile | $2.99 |

**속도:** 평균 10~30초 (사람이 풀어줌)

**주의:** 인터파크 CAPTCHA = 일반 이미지 → $1/1000개

### 3. 프록시 (가격 비교)

| 서비스 | 가격/GB | 한국 IP | 특징 |

| 감지 방법 | 우회 방법 | 우리 대응 |
|-----------|----------|----------|
| 새로고침 과다 | 랜덤 딜레이 | ✅ 구현 예정 |
| WebDriver 플래그 | Camoufox | ✅ 자동 처리 |

| 안심예매창 (퍼즐) | 수동 or 2Captcha | ⚠️ 수동 필요할 수도 |
| **얼굴인식 (예정)** | 불가능 | ❌ 아직 미도입 |

### 최악의 시나리오
1. **계정 정지** → 백업 계정 사용
2. **IP 차단** → 프록시 교체 (1초)
3. **CAPTCHA 무한** → 수동 입력 전환
4. **서버 다운** → 모두 같은 조건 (공정)
